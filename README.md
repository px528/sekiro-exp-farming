# sekiro-exp-farming🗡️

只狼：影逝二度 自动刷分脚本，基于 `pydirectinput` 模拟键鼠操作，保留运行日志。

## 功能

- 自动循环执行击杀流程（R → ↓ → Enter → 移动/攻击/闪避）
- 保留详细 `print` 日志，实时显示按键与进度
- 启动前 3 秒倒计时，方便切换到游戏窗口
- 代码精简，使用短函数名（`h` 按住，`k` 按下）

## 依赖

```bash
pip install pydirectinput
```

## 使用

1. 启动游戏，进入可刷分场景
2. 运行脚本：`python sekiro_farmer.py`
3. 3 秒内切到游戏窗口
4. 脚本自动循环指定次数后结束，控制台显示进度

## 配置

在脚本顶部修改：

```python
LOOP = 1000   # 循环次数
D = 0.15      # 按键间基础间隔（秒）
```

## 代码结构

- `h(key, duration)`：按住按键指定秒数，带日志
- `k(key, times=1)`：按下并释放按键，带日志
- 主循环：依次执行击杀流程
