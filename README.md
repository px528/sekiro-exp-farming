# sekiro-exp-farming🗡️

只狼：影逝二度 自动刷分脚本，基于 `pydirectinput` 模拟键鼠操作，保留运行日志。

## 性能

每个循环36秒，获得391经验，和用户手动刷相比效率很低，不过胜在可以稳定挂机。
每小时刚好100个循环，获得39100经验，一晚上挂机8小时可得312800经验。

## 功能

- 自动循环执行击杀流程（归佛->击杀->归佛->击杀->归佛->击杀->...）
- 保留详细 `print` 日志，实时显示按键与进度
- 启动前 3 秒倒计时，方便切换到游戏窗口
- 代码精简，使用短函数名（`h` 按住，`k` 按下）

## 依赖

```bash
pip install pydirectinput
```

## 使用

1. 启动游戏，进入 天守阁上层 苇名流道场（击杀巨型忍者枭以后）
2. 坐鬼佛确保归佛能回到这里 
2. 运行脚本：`python sekiro-exp-farming.py`
3. 3 秒内切到游戏窗口
4. 脚本自动循环指定次数后结束，控制台显示进度
